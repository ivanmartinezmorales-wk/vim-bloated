# vim-bloated

The ultimate vim bloatware for ultrafast nvim development.

## About

Comes with goodies to make you code at the speed of 200ms. Contains goodies like:

- vim-sensible
- vim-surround
- coc.nvim
- vim-airline
- vim-airline-themes
- vim-polyglot
- junegunn/fzf
- junegunn/fzf.vim
- markdown-preview
- auto-pairs

Loads super fast. Loads at around 200ms on a 2013 Macbook Pro.
